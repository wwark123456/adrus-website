# ADRUS Website

A polished static website for ADRUS that is ready to upload to GitHub and publish with GitHub Pages.

## Files

- `index.html` - main website page
- `styles.css` - site styling
- `script.js` - mobile menu and demo form behaviour
- `assets/adrus-logo.png` - ADRUS logo used across the site

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. Commit the files.
4. Go to **Settings** -> **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root` folder.
7. Save.
8. GitHub will generate a live website link.

## Connect your domain

1. In GitHub Pages settings, add your custom domain.
2. In your domain provider, point the domain DNS records to GitHub Pages.
3. Wait for DNS propagation.
4. Enable HTTPS in GitHub Pages once available.
